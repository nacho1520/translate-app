# Translate App

A Translate App page made with Next JS and Tailwind. 

## Technologies:

* Next JS
* React JS
* Tailwind

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
